## 📋 What is asc-screenshots?

asc-screenshots is a simple tool that helps you make professional screenshots for iPhone and iPad apps. It prepares images that are ready to use in the Apple App Store. The tool exports your work in a format compatible with asc-client, which is a common uploader for the App Store. You don’t need special skills to use it.

## ⚙️ How to Use asc-screenshots

Follow these simple steps once the program is running:

1. Launch the software by double-clicking `asc-screenshots.exe`.  
2. On the home screen, select the device type (iPhone or iPad) for your screenshots.  
3. Upload your app images by clicking the **Add Images** button.  
4. Arrange the images in the order you want them to appear on the App Store.  
5. Use the preview area to check how your screenshots will look.  
6. Click **Export** to save the screenshots in the asc-client format.  
7. Choose a folder to save the exported files on your computer.

## ❓ Troubleshooting Tips

If you encounter issues, try these solutions:

- The program won’t start: Make sure you installed all parts and run `asc-screenshots.exe` as administrator.  
- Images do not appear: Check that your picture files are in a supported format like PNG or JPEG.  
- Export fails: Ensure you have permission to save files in the chosen folder. Try selecting a different folder if needed.  
- Screenshots are the wrong size: Confirm you selected the right device type before exporting.  

If problems continue, you can open a new issue on the GitHub page for support.
