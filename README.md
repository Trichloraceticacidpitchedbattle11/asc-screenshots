# asc-screenshots

> **This skill has moved to [ascelerate](https://github.com/keremerkan/ascelerate).** This repository is no longer maintained.

The app-store-screenshots skill is now integrated into [ascelerate](https://github.com/keremerkan/ascelerate) as a built-in skill. Install it with:

```bash
npx skills add keremerkan/ascelerate
```

Or for Claude Code specifically:

```bash
ascelerate install-skill
```

See the [ascelerate README](https://github.com/keremerkan/ascelerate#using-with-app-store-screenshots) for full documentation.
